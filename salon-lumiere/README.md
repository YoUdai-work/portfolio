# 制作実績｜隠れ家フェイシャルエステ「Lumière」LP（架空案件）

> ポートフォリオ用デモ。美容サロンの新規集客を想定したランディングページ。

## 想定課題
30〜40代女性向けの個室エステ。「エステ＝勧誘がこわい」という心理的ハードルで新規予約に至らない、という想定課題を設定。

## 狙い（CVR設計）
- **ファーストビューに「価格つきCTA」を配置** … 「初回体験¥4,980を予約」と最初の画面で行動と価格を提示し、離脱前に意図を伝える。
- **悩み共感 → 選ばれる理由 → 料金 → お客様の声 → 予約** の王道導線 … 不安を先に解消してから申込へ。
- **「しつこい勧誘なし」を繰り返し明示** … 想定ターゲットの最大の不安を直接つぶす。
- **スマホ固定の予約ボタン** … 美容LPはスマホ流入が大半。常時CTAを画面下に固定し予約率を底上げ。

## デザイン意図
- **配色**：ベージュ×ゴールド×くすみピンク。高級感・清潔感・リラックスを想起させ、ターゲット層の感性に合わせた。
- **書体**：見出しに明朝（Noto Serif JP）で上質さ、本文はゴシックで可読性を確保。
- **余白とアニメ**：余白を広く取り「丁寧で落ち着いた」ブランド印象。スクロールで要素をふわっと表示し体験の質を上げた。

## 技術
- HTML/CSS/JS（単一ファイル・依存ライブラリなし）。レスポンシブ対応。
- IntersectionObserverでスクロールアニメ、FAQアコーディオン、固定CTAを純JSで実装。

## 確認方法
- ローカル：`index.html` をブラウザで開くだけ。
- 公開：Vercel / Netlify / GitHub Pages にこのフォルダを上げるとURLで共有可能。
